Race To The End

A simple, yet challenging datapack that tasks teams of 3 to beat Minecraft.

 Download
Description

Teams of 3 will battle it out in a race to the end. The team that kills the ender dragon first wins the game. You can sabotage other teams, focus on getting to the end to win or you can wait till the time is right to steal victory from your opponents. Everyone has 3 lives, if you use them all up, you're out, and its up to your teammates to finish the job, help them out in spectator mode!

Features
people_alt

Teams of 3

The game is designed to support 3 teams to battle to the end. This will be optional in a future update.

 filter_alt

Sectioned game

The game starts off with a grace period to allow people to gather resources and not have the game turn into a mass fight.

 favorite

Player lives

Each player spawns with 3 lives and once they are out, they will be moved into spectator mode.

 accessibility_new

Easy to use

All events in the UHC can be easily configured and changed to your liking using the global scoreboard.

Installation

As with every datapack, there is a very simple installation process.

  1. Click the download button above then save it in your Downloads.
  2. Open your singleplayer menu, press Create New World then Data Packs
  3. Drag the .zip file from your Downloads into the available space on the left.
  4. Click the right arrow button and move it over (ignore any version errors - it works fine)
Setup

Once the datapack is installed, you can start the game easily.

  1. If you have any options you want to tweak, edit them now through the global scoreboard
  2. When everything is ready, you can run function racetotheend:start and let the games begin!
