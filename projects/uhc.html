Description

A simple, lightweight and easy-to-use UHC datapack designed for 1.16.5 (may backport to older versions). Featuring popular UHC standard additions like Cut Clean and Speed UHC, this is the easiest and simplest way to hold a UHC in latest version on platforms like Minecraft Realms.

Features
update

Customisable timings

Using scoreboards you can configure event timings to make the event as slow or fast as you want!

 more_time

Cut Clean

When iron ore and gold ore is mined or animals are killed, the item drops are instantly smelted and ready-to-go.

 filter_alt

Sectioned game

The UHC is split into 3 main sections; the starter period, grace period and main game. More information here.

 speed

Speed UHC

Any tool you hold will automatically be enchanted with Efficiency 1 to speed up mining.

 no_accounts

Dead spectators

When the grace period has ended, dead players are automatically moved into spectator mode to prevent interference.

 group

Simple teams

Players can easily be assigned either Red or Blue team (team1 and team2) to make the game more competitive.

 accessibility_new

Easy to use

All events in the UHC can be easily configured and changed to your liking using the global scoreboard.

 favorite

Natural healing

After the grace period has ended, naturalRegeneration is disabled meaning you can only heal health with golden apples & potions.

Installation

As with every datapack, there is a very simple installation process.

  1. Click the download button above then save it in your Downloads.
  2. Open your singleplayer menu, press Create New World then Data Packs
  3. Drag the .zip file from your Downloads into the available space on the left.
  4. Click the right arrow button and move it over (ignore any version errors - it works fine)
Setup

Once the datapack is installed, you can start the game with one simple command.

  1. To ensure everything is loaded correctly, run function uhc:defaults - this will revert all settings to default
  2. If you have any options you want to tweak, edit them now through the global scoreboard
  3. When everything is ready, you can run function uhc:start and let the games begin!
  4. If you want to pause (or stop) the events temporarily, you can set the timer_enabled score in the global scoreboard to 0.

Other options (timings for starting period & grace period, spectator deaths, teams mode etc.) can be configured in the global scoreboard.

  • In solos mode, the last player standing will win the game
