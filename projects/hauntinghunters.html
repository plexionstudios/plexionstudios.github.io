The Grand Scheme of Things Hide & Seek is a unique take on a classic, featuring unique powerups! You spawn outside a mansion and, as a seeker, have to find your way to victory, killing all the hiders.

group

Multiplayer

By using Minecraft Realms or hosting a server, you can invite as many players as you want to play as either a hider or a seeker.

Powerups!

Before the game starts, you are given a set of gold to purchase a range of powerups to affect the opposing team or help you in a close situation.

Installing a Map into Minecraft is very simple, as follows:

  1. Click the download button above then save it in your Downloads.
  2. Press WIN + R which should open a run box, then type %appdata% and press ENTER
  3. Navigate to .minecraft then saves and drag the .zip file inside.

Hosting a server

  1. Pick up the latest Minecraft server.jar at minecraft.net then launch it.
  2. The program should close, afterwhich find a text file in the same directory called eula.txt - open the file and replace false with true (accepting the EULA)
  3. Extract the Map.zip then copy the folder into the server directory.
  4. Rename the folder to world
  5. Set-up port forwarding for 25565 on your router, more information on different routers here: noip.com
  6. Launch up your server and share the IP with your friends!

Using Minecraft Realms

  1. Ensure the map can be found in your Singleplayer menu.
  2. Then, navigate to Minecraft Realms and click the build button.
  3. You should be presented with 3 world slots, choose one and then select Reset world and Upload world
  4. Locate the world in the menu and select it.
  5. Once finished, you can invite players to your realm and have some fun!
