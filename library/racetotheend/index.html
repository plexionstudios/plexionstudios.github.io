Race To The End

A simple, yet challenging datapack that tasks teams of 3 to beat Minecraft.

Download latest View on PMC

Overview

Teams of 3 battle it out in a race to the end. The team that kills the ender dragon first wins the game. You can sabotage other teams, focus on getting to the end to win or you can wait till the time is right to steal victory from your opponents. Everyone has 3 lives, if you use them all up, you're out, and its up to your teammates to finish the job, help them out in spectator mode!

Installation & Setup

As with any other datapack, there is an incredibly simple installation process:

  1. Download the datapack .zip file and save it somewhere
  2. In Minecraft, create a new world then press Data Packs
  3. Drag the file you saved earlier into this now opened folder
  4. Then click the > arrow to enable the pack, ignore any version errors

Now the pack is installed, there is a simple process to start the game:

  1. Either host the server or load up the world on Minecraft Realms
  2. Once you're ready, run /function uhc:start and let the games begin!
