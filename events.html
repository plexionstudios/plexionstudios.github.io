The benefits of Event Datapacks

All Event Datapacks listed have many benefits over the competition.

thumb_up

To be trusted

Each datapack has been tested before release and on my channel so you can know that the datapack won't fail on you. If it does happen to fail, which is very rare, simply contact me via PMC, YouTube replies or by email @ plexionbusiness@gmail.com and i'll get the problem fixed as soon as possible.
accessibility_new

Ready in seconds1

Datapacks are very easy to install and setup, with only a few clicks required. Every datapack also features customisable options via scoreboards to allow further customisation for your event. This allows all events to be fully controllable incase there's something you want changed.

More information
report_problem

Performance is key

All datapacks are optimised to reduce client-side lag and server-side lag. However, this does come with some consequences, mainly with falling blocks which are usually killed in most datapacks to prevent lag. Currently, these features are not optional and are hard-coded into the events.
