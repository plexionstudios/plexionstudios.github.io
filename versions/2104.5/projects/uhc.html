Description

A simple, lightweight and easy-to-use UHC datapack designed for 1.16.5 (may backport to older versions). Featuring popular UHC standard additions like Cut Clean and Speed UHC, this is the easiest and simplest way to hold a UHC in latest version on platforms like Minecraft Realms.

Features
update

Customisable timings

Using scoreboards you can configure event timings to make the event as slow or fast as you want!

 more_time

Cut Clean

When iron ore and gold ore is mined or animals are killed, the item drops are instantly smelted and ready-to-go.

 filter_alt

Sectioned game

The UHC is split into 3 main sections; the starter period, grace period and main game. More information here.

 speed

Speed UHC

Any tool you hold will automatically be enchanted with Efficiency 2 to speed up mining.

 no_accounts

Dead spectators

When the grace period has ended, dead players are automatically moved into spectator mode to prevent interference.

 group

Simple teams *Green Team (team3) added in 1.0.4

Players can easily be assigned either Red or Blue team (team1 and team2) to make the game more competitive.

 accessibility_new

Easy to use

All events in the UHC can be easily configured and changed to your liking using the global scoreboard.

 favorite

Natural healing *Removed in 1.0.4

After the grace period has ended, naturalRegeneration is disabled meaning you can only heal health with golden apples & potions.

Installation

As with every datapack, there is a very simple installation process.

  1. Click the download button above then save it in your Downloads.
  2. Open your singleplayer menu, press Create New World then Data Packs
  3. Drag the .zip file from your Downloads into the available space on the left.
  4. Click the right arrow button and move it over (ignore any version errors - it works fine)
Setup

Once the datapack is installed, you can start the game easily.

  1. If you have any options you want to tweak, edit them now through the global scoreboard
  2. When everything is ready, you can run function uhc:start and let the games begin!

If you want to pause (or stop) the events temporarily, you can set the timer_enabled score in the global scoreboard to 0.

Updates
Version 1.0.4Published 21/02/2021

This update is an optional side-branch to 1.0.3; updating is not necessary

  • There are now 3 teams, red, blue and green
  • The world border will now shrink considerably less
  • naturalRegeneration is now enabled throughout the game
  • The difficulty is now set to peaceful in pre-game then set to normal going forward
  • If a player gets a kill, they will be given a gapple as a reward
  • Apple spawn rates have been reduced
  • Team victories are now more specified in the code
Version 1.0.3Published 17/02/2021

This update enforces gamemode & effects onto all new players joining

  • Pre-game is now set as period -1
  • Victory splash screens are now set as period 3
  • Victory splash screens are now split into 3 functions, solos_victory, red_victory and blue_victory
  • Players now have a lastlogin scoreboard entry displaying the last period they were logged on for
Version 1.0.2Published 15/02/2021

This update brings a crucial gamerule change & some teams additions:

  • naturalRegeneration is now enabled for the first 2 phases, but disabled for the rest of the game
  • Loot tables for apples mined from Oak Leaves have been boosted
  • Teams mode is now an option in the global scoreboard, listed as enable_teams
  • In teams mode, if the other team is fully eliminated the last remaining team will win
  • In solos mode, the last player standing will win the game
Version 1.0.1Published 14/02/2021

This update brings along one addition with Cut Clean and a tweak to the border:

  • All animal food drops are instantly smelted
  • The world border will now close in to 50 blocks in the main game period
