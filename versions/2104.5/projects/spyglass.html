Spyglass HUD Remover

A simple pack designed for version 20w45a+ of 1.17 that removes the circular border you see when scoping with the Spyglass.

 More info Download
Description

A texture pack designed for snapshot 20w45a+ (not tested in future versions) that removes the circular border shown when using the new Spyglass item. The same effect can be achieved in vanilla by using F1 or using OptiFine zoom but this makes the task easier.

Gallery
Installation

Resource packs are the simplest thing to install on Minecraft. Here are the simple steps.

  1. Click the download button above then save it in your Downloads.
  2. Press WIN + R then type %appdata%
  3. Navigate to .minecraft then resourcepacks and drag the .zip file inside.
  4. Open Minecraft, then go to Options... then Resource Packs... and find the pack in there.
  5. Apply the pack with the > button and press Done
