How to install

You can install PlexFonts by with three simple steps.

  1. Navigate to PlexFonts and click "See inside" in the top-right.
  2. Drag the "PlexFonts" sprite into your backpack.
  3. Navigate to PlexFonts and click "See inside" in the top-right.
Functionality of PlexFonts

PlexFonts can be used to display any text in the English language using the main symbols present (like the alphabet). You can have a dive into the costumes tab to see the full extent of what it supports. If you want a character or language supported, it's probably on my list but feel free to drop me a message on Scratch or by email!

Custom colouring & brightness tweaking

With the latest version of PlexFonts, you can colour text & modify brightness to the full extent of Scratch. This can all be done with the $FontColour variable.
