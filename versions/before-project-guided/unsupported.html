Unsupported projects

Learn about what an unsupported project is, how to spot if one is unsupported and why it's unsupported.

What does it mean if a project is unsupported?

When a project is unsupported, no more updates will be pushed to the page and it will be left basically archived. Overtime, the project will eventually become obsolete and not function.

Why is a project generally unsupported?

A project can be discontinued for a variety of reasons:


It has been replaced.

The project you are looking at may have been replaced with a better made version. In this case, a notice should be placed somewhere in the page.


It's no longer needed.

The project may have 'expired' and is no longer useful. This is the case a lot of the time for countdowns to specific dates when the time has already passed.
